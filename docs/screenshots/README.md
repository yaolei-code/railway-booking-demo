# 项目截图

启动项目后截取以下页面，保存到此目录，README 会自动展示。

## 需要截取的页面

1. **ticket-search.png** — 查票下单页（输入起止站和日期，展示车次列表）
2. **orders.png** — 我的订单页（已支付 / 待支付 / 已取消）
3. **account.png** — 登录注册页
4. **admin.png** — 管理台（车站 / 车次 / 库存管理）
5. **knife4j.png** — Knife4j 接口文档页面（http://localhost:8080/doc.html）

## 启动项目

```bash
docker compose up -d
```

然后访问 http://localhost:5173 依次截取各页面。
